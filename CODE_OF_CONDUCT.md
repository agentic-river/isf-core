# Community Code of Conduct

## Our Pledge
We are committed to making participation in the Infinite Software Factory (ISF) community—including reporting bugs, proposing features, and participating in public discussions—a welcoming and harassment-free experience for everyone.

We pledge to act and interact in ways that contribute to an open, welcoming, diverse, inclusive, and healthy community.

## Standards of Behavior
We expect all community members to:
* Be respectful and considerate of other members and maintainers.
* Use welcoming and inclusive language.
* Gracefully accept constructive feedback.
* Focus on what is best for the overall community.

Behavior that is unacceptable includes:
* Harassment, trolling, insulting/derogatory comments, or personal attacks.
* Spamming public issue trackers or feedback channels.
* Publishing others' private information without explicit permission.
* Other conduct which could reasonably be considered inappropriate in a professional setting.

## Scope
This Code of Conduct applies to all public community spaces associated with ISF, including GitHub Issues, GitHub Discussions, and community forums.

Since ISF is a closed-source product, direct code contributions and Pull Requests are not accepted. Public interaction is focused entirely on user support, product feedback, and community discussions.
