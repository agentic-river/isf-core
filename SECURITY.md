# Security Policy

## Reporting a Vulnerability

We take the security of the Infinite Software Factory (ISF) and its scaffold environments seriously. Because ISF is designed to run locally on your filesystem and interface with Docker containers, protecting your workspace integrity is a top priority.

If you discover a security vulnerability, please **DO NOT** open a public issue. Doing so exposes users to potential exploits before a patch can be deployed.

Instead, please report vulnerabilities by emailing **security@agenticriver.com** or reaching out to the organization maintainers directly.

We will acknowledge your report within 48 hours and provide a status update on remediation efforts within 5 business days. Thank you for helping keep our developer community safe!
